# Fcmam5's dotfiles

Automatizing the boring step when moving to a new computer the Installation & configuration, simply running:

```shell
sudo chmod 777 setup.sh
sudo ./setup.sh
```
Will do the following:
* Update the system
* Install
* ...
* Configure
* Cleanup after the installation
